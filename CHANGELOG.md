# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-05-09

### Added

- Initial production-safe minimal release.
- Attribute-based encryption with `#[Encrypted]`.
- Halite-based field encryptor.
- Symfony bundle configuration for `key_file`, `auto_generate_key` and `allow_plaintext`.
- Explicit key generation and validation commands.
- Package-specific exception hierarchy.
- Doctrine ORM lifecycle integration for transparent field encryption.
- Readonly encrypted property validation.
- Unit and integration tests for key handling, plaintext policy, command behavior and Doctrine ORM integration.

### Changed

- Missing key files no longer generate a key silently by default.
- Plaintext values in encrypted fields are rejected by default.
- Internal metadata, key and subscriber classes are marked as internal API.

### Security

- Key files generated by the library are written with `0600` permissions.
- The README documents key backup requirements, plaintext migration mode and lifecycle-event limitations.
